# Iconos de la Extensión

La extensión necesita iconos en los siguientes tamaños:

- **16x16** - Icono en la barra de herramientas
- **48x48** - Icono en la página de gestión de extensiones
- **128x128** - Icono para Chrome Web Store

## Formato

- PNG con transparencia
- Fondo transparente recomendado
- Diseño simple y reconocible

## Diseño sugerido

Un icono de impresora térmica con:
- Forma rectangular (ticket saliendo)
- Colores: azul (#228be6) y blanco
- Estilo minimalista

## Generar iconos

Puedes usar herramientas online como:
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
- Canva
- Figma

O crear manualmente con:
- GIMP
- Photoshop
- Inkscape (SVG → PNG)

## Iconos placeholder

Por ahora, la extensión funcionará sin iconos (Chrome usará uno por defecto).
Para producción, se deben crear los iconos profesionales.
